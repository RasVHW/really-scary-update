# The Really Scary Update
![reallyscaryupdate.png](src%2Fmain%2Fresources%2Freallyscaryupdate.png)

An update for BTA 7.1, based on the release 1.4.2 Pretty Scary Update

## Dependencies

HalpLibe >= 3.5.2

ModMenu >= 2.0.5

DragonFly >= 1.4.4

## Implemented Features
* White Pumpkin
  * Regular
  * Glowing
  * Slashed
  * Glowing Slashed
